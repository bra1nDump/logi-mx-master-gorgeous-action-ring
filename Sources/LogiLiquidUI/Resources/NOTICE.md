# OpenAI mark

`OpenAIChatGPTMark.svg` contains the unmodified Blossom/ChatGPT mark geometry
from OpenAI's official brand asset, cropped only to a square transparent canvas
for use as the icon of the action that directly invokes ChatGPT.

Source: https://openai.com/brand/
