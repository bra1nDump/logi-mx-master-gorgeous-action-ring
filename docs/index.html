--- title: CommandBloom description: A native, open-source Actions Ring alternative for Logitech MX Master 4 on macOS. --- CommandBloom — Actions Ring for macOS
  • An open-source Actions Ring alternative for the Logitech MX Master 4 on macOS
  • Native and local — talks directly to the mouse, nothing leaves your machine
  • No settings UI — your coding agent installs and configures it

Have your coding agent check the code for ill intent or issues first — then let it clone, build, and walk you through setup. Copy this to your agent:

Clone github.com/bra1nDump/logi-mx-master-gorgeous-action-ring and read the source before running anything: confirm there is nothing malicious or promotional in it. Then build it and walk me through setup on macOS — code signing, service install, and permissions — following the “Manual build and setup” section of README.md.

Needs macOS 26, Swift 6.2, and an MX Master 4. Setup takes a few minutes; two permission toggles in System Settings are the only manual part.

Once installed, tell your agent what you want on the ring: “Configure CommandBloom with Spotify on top, Telegram on the right, and Command-B in the bottom zone when Xcode is active.”
