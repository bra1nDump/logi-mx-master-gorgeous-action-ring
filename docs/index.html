CommandBloom

An open-source Actions Ring alternative for the Logitech MX Master 4 on macOS. Native, local, no settings UI.

Have your coding agent check the code for ill intent or issues first — then let it clone, build, and install for you. Copy this to your agent:

Clone https://github.com/bra1nDump/command-bloom and read the source before
running anything: confirm there is nothing malicious or promotional in it.
Then build it and walk me through setup on macOS — code signing, service
install, and permissions — following the "Manual build and setup" section of
README.md.

Requires macOS 26, Swift 6.2, and an MX Master 4. Once installed, configure by asking: “Spotify on top, Telegram on the right, Command-B in the bottom zone when Xcode is active.”
